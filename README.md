## Overview
This is the public-facing documentation and support repository for the devjob.plan app for macOS.  Please visit the [product site](https://realprovenllc.github.io/devjob.plan/) for more details or submit an [issue](https://github.com/RealProvenLLC/devjob.plan/issues) here for support requests.

#### Notes
- This site is built with [Hugo](https://gohugo.io/) and styled with the most excellent [Hextra theme](https://themes.gohugo.io/themes/hextra/).
- The Hugo theme is attached using git submodules, so be sure you `git submodule update --init` after cloning.
